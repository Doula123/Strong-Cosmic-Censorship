Welcome

This prediction tool will use a trained AI model to predicted the planetery disposition of observed objects of interest. It uses a random trees algorithm to classified each object as a confirmed planets, candidate planets, or false positives.

Your CSV file should follow these guidelends:

  • Contain column: "planet_name"
  • Contain at least three of the following columns:
    • Column: "model_snr"
    • Column: "planet_rad"
    • Column: "depth"
    • Column: "impact"
    • Column: "orb_period"
    • Column: "duration"
  • May contain optional columns:
    • Column: "planet_eq_temp"
    • Column: "stellar_teff"
    • Column: "planet_insol"
    • Column: "stellar_rad"
    • Column: "stellar_g_log"
  • Dataset does not need to be complete, empty spaces will be accounted for
